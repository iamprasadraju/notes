A linked list is ==a fundamental linear data structure made of connected **nodes**, where each node holds **data** and a **pointer (link)** to the next node, forming a chain not necessarily stored in contiguous memory, allowing for efficient dynamic size changes and insertions/deletions compared to arrays==.


<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/6/6d/Singly-linked-list.svg/1280px-Singly-linked-list.svg.png" alt="linked list"> 


A linked list is a sequence of nodes that contain two fields: data (an integer value here as an example) and a link to the next node. The last node is linked to a terminator used to signify the end of the list.
