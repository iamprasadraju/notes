**How BFS Works ?**

1. it starts at node.
2. First visits all its immedite neighbours
3. Then visits the neighbors' neighbors.
4. Then the next level and so on...

Key Points about BFS

- Uses a Queue (FIFO)
- Finds shortest path
- Uses more memory
